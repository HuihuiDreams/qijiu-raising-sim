# 上传代码到 GitHub

// turbo-all

## 步骤

1. 查看当前修改状态：

```bash
git status
```

1. 添加所有修改的文件：

```bash
git add .
```

1. 提交更改（根据实际修改内容自动生成提交信息）：

```bash
git commit -m "更新代码"
```

1. 推送到 GitHub：

```bash
git push origin main
```

## 完成

推送完成后，代码已同步到：<https://github.com/HuihuiDreams/qijiu-raising-sim>
